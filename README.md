# vue-recruit

这是招聘系统vue3前端代码，[数据库以及后端传送门](https://github.com/yanqing8/spring-recruit)。如果能帮到你可以帮忙点个小星星吗？

## 配置

node.js 16获更高。

建议使用pnpm管理器 
```sh
npm install -g pnpm
```


## 安装依赖

```sh
pnpm install
```

### 启动程序

```sh
pnpm dev
```

### Compile and Minify for Production

```sh
pnpm build
```

### Lint with [ESLint](https://eslint.org/)

```sh
pnpm lint
```
